# Agent Manager Specification Overview

## 概要
AIエージェントによるスクラム開発プロセスを自律的に実行する管理システム。AIはGitHub Copilot SDKを用いて組み込み、成果物やコンテキストはgit管理。スプリント期間は設けず、AIがレビュータイミングでスプリントをプランし、人間はスプリントレビューで成果物を評価する。

## 主な特徴
- プロダクト毎にバックログアイテム管理
- スクラム開発手法に則ったバックログアイテム分割・管理
- AIエージェントの役割管理（PO, SM, Dev, Reviewer, QA）
- バックログアイテム分割・協調作業・レビュー・品質保証をAIが自律実行
- コンテキスト・成果物はgit管理（mcp-server-git利用）
- Copilot SDKによるAI組み込み（将来的に複数AI対応）
- スプリント期間はAIがレビュータイミングで自動設定
- 人間はスプリントレビューで成果物評価
- 成果物はGitHubにpush

## システム構成
- バックエンド：バックログ管理、AI連携、ワークフロー
- フロントエンド：UI/UX（バックログ、スプリント、レビュー画面）
- データベース：プロダクト・バックログアイテム・AIエージェント情報管理
- AI連携：Copilot SDK、mcp-server-git
- GitHub連携：成果物push、PR、レビュー

## 拡張性
- エージェント抽象化により将来的なAI追加が容易
- スクラム手法に則った自動バックログアイテム分割・協調作業
